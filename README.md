# Tugas Besar Struktur Data

## Topik: Pengembangan Game Berbasis Graph

### Deskripsi Program: Program ini merupakan aplikasi simulasi permainan sederhana berbasis struktur data Graph untuk logika pencarian rute. Pengguna harus memanajemen Energi (Cost) dalam mencapai tujuan, serta dilengkapi dengan 3 Tingkat Kesulitan (Bayi, Remaja, Sepuh) yang menguji efisiensi langkah pengguna dalam peta yang saling terhubung.
